# Linux Basics Assignment

This repository contains my hands-on practice and solutions for fundamental Linux commands and operations. The assignment focuses on essential skills required to work efficiently in a Linux environment.

---

## 📚 Topics Covered

### 1. Creating and Renaming Files/Directories
- Created directories using `mkdir`
- Created files using `touch`
- Renamed files using `mv`

---

### 2. Viewing File Contents
- Displayed file contents using `cat`
- Viewed first few lines using `head`
- Viewed last few lines using `tail`

---

### 3. Searching for Patterns
- Used `grep` to search specific keywords within files

---

### 4. Zipping and Unzipping
- Compressed directories using `zip`
- Extracted files using `unzip`

---

### 5. Downloading Files
- Downloaded files from the internet using `wget`

---

### 6. Changing Permissions
- Modified file permissions using `chmod`
- Set files to read-only mode

---

### 7. Environment Variables
- Created and used environment variables with `export`

---

## 🚀 Objective

The objective of this assignment is to build a strong foundation in Linux by practicing commonly used commands and understanding their real-world usage.

---

## 🧠 Key Learnings

- Basic file and directory management
- File content handling and navigation
- Searching and filtering data
- File compression techniques
- Permission management
- Working with environment variables
