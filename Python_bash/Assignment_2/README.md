## ✅ Q1. Grade Checker

### 📌 Description

A program that takes student marks as input and assigns a grade based on predefined conditions.

### 🎯 Concepts Used

* `if-elif-else`
* User input
* Conditional logic

### 📊 Grading Criteria

| Marks Range | Grade |
| ----------- | ----- |
| 90+         | A     |
| 80–89       | B     |
| 70–79       | C     |
| 60–69       | D     |
| Below 60    | F     |

## ✅ Q3. Write to a File

### 📌 Description

A program to create a file and write content into it using file handling.

### 🎯 Concepts Used

* `open()` function
* File modes (`a+`)
* `write()`
* `readlines()`
* File pointer (`seek()`)

### 💡 Key Learning

* Append mode (`a+`) adds content without deleting existing data

---

## ✅ Q4. Read from a File

### 📌 Description

A program to read and display content from a text file.

### 🎯 Concepts Used

* File reading (`r` mode)
* `read()` function

### 💡 Key Learning

* `"r"` mode is used only for reading existing files

## ▶️ How to Run

### 🔹 Run any Python file:

### 🔹 Example:

```

python reading_file.py
```
